# 🛡️ Machine Learning Project — Phishing Email Detection

A Machine Learning project focused on detecting phishing emails using Natural Language Processing (NLP) and classification algorithms. This project aims to analyze email content and metadata to automatically classify emails as phishing or legitimate.

---

## 📌 Project Overview

Phishing email attacks are one of the most common cybersecurity threats used to steal sensitive information such as login credentials, banking data, and personal information. Manual detection becomes increasingly ineffective as email volume grows rapidly.

This project applies Machine Learning techniques to build an automated phishing email detection system based on:

* email body content
* subject text
* URLs
* email metadata

---

## 🎯 Objectives

* Perform exploratory data analysis (EDA) on phishing email datasets
* Build a clean NLP preprocessing pipeline
* Extract meaningful text features
* Train classification models for phishing detection
* Explore clustering and anomaly detection approaches

---

## 📂 Dataset

Dataset source:

* Kaggle — Phishing Email Dataset

Dataset consists of multiple public email corpora:

* Enron
* CEAS_08
* Ling
* Nazario
* Nigerian Fraud
* SpamAssassin

Approximate dataset size:

* 82,000+ emails
* phishing and legitimate labels
